About

This site is built with plain HTML and CSS, structured using Fibonacci Web Design principles — golden ratio proportions applied recursively to layout, typography, and spacing.

All measurements derive from the Fibonacci sequence with a base unit of 2px: 2, 4, 6, 10, 16, 26, 42, 68, 110, 178, 288, 466, 754. The two-column home layout uses a 466/288 split, where 288/466 ≈ 0.618 = φ. See the design spec for details.
